*****The Properties of a Transition - create a transition*****

.style {
  /*...(more styles)...*/
  transition-property: all;
  transition-duration: 1s;
  transition-timing-function: ease;
  transition-delay: 0s;
}

----------------------------------------------------------

*****Fun Transitions for Responsive Websites*****

* {
  transition: all 1s;
}
